Appendix A
REDHAWK Yum Repository and Packages

The following sections describe the REDHAWK packages and provided dependencies.

External dependencies are also necessary for development with REDHAWK and for building REDHAWK from source. Refer to Appendix B.

A.1 REDHAWK Yum Repository

All of the REDHAWK packages are included in a yum repository that can be set up using the process described in Section 2.2.2.2. The repository contains three yum groups (REDHAWK Runtime, REDHAWK Development, and REDHAWK Enterprise Integration) and additional OS dependencies. The yum groups and additional OS dependencies are described in the following sections:

A.1.1 REDHAWK Yum Groups

A.1.1.1 REDHAWK Runtime
